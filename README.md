# 截图提交微信小程序

## 项目简介

这是一个可以提交截图的微信小程序，主要用于普通用户进行工作汇报。用户可以上传截图，添加描述，选择分类，并直接提交。

## 核心功能

- 截图上传：支持从相册选择或拍照上传
- 文字描述：添加截图相关的文字说明
- 分类管理：支持对提交的截图进行分类
- 直接提交：无需审核，直接提交成功
- 历史记录：查看历史提交记录
- 数据管理：使用微信云开发存储数据

## 技术栈

- 前端：原生微信小程序框架
- 后端：微信云开发（无需自建服务器）
- 数据存储：微信云数据库
- 文件存储：微信云存储

## 快速开始

### 1. 准备工作

- 注册微信小程序账号：https://mp.weixin.qq.com/
- 安装微信开发者工具：https://developers.weixin.qq.com/miniprogram/dev/devtools/download.html

### 2. 配置云开发环境

1. 在微信开发者工具中打开项目
2. 点击左侧导航栏的"云开发"按钮
3. 创建一个新的云开发环境
4. 记录云开发环境ID

### 3. 修改配置

1. 打开 `app.js` 文件，将 `env` 字段修改为你的云开发环境ID

```javascript
wx.cloud.init({
  env: 'your-cloud-env-id', // 替换为你的云开发环境ID
  traceUser: true
})
```

### 4. 部署云函数

1. 在微信开发者工具中，右键点击 `cloudfunctions/init` 目录
2. 选择"上传并部署：云端安装依赖"
3. 等待部署完成

### 5. 初始化数据库

1. 在微信开发者工具中，点击"云开发"按钮进入云开发控制台
2. 点击"云函数"标签页
3. 找到 `init` 函数，点击"测试"按钮
4. 执行测试，初始化数据库和默认分类

### 6. 运行小程序

1. 在微信开发者工具中，点击"编译"按钮
2. 可以在模拟器中查看和测试小程序
3. 也可以点击"预览"按钮，使用微信扫描二维码在手机上查看

## 注意事项

- 本项目使用微信云开发，需要确保云开发环境已正确配置
- 首次使用时，需要执行 `init` 云函数来初始化数据库和默认分类
- 小程序需要获取相册和相机权限，以便选择和拍摄图片
- 云开发环境有一定的免费配额，超出配额后可能需要付费

## 性能优化

- 图片上传前会进行压缩，减少上传时间和存储空间
- 使用云开发的CDN加速，提高图片加载速度
- 数据查询使用索引，提高查询效率
- 页面渲染使用按需加载，减少初始加载时间
